> 清空成员表

```SQL
db.collection('TB_MEMBER')
.where({
  _id: _.exists(true)
})
.remove()
```

### 状态码

| 微信 (code / errCode ) |   报错信息 (errMsg)    |
| :--------------------: | :--------------------: |
|           -1           | 数据库内未找到相关信息 |

| 自定义 (code / errCode) | 报错信息 (errMsg) |
| :---------------------: | :---------------: |
|           200           |       正常        |
|         500001          |   密令校验非法    |
|         500002          |   重复注册用户    |
|         500003          |   注册昵称重复    |
